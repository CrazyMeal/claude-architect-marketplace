# Documentation Assistance

Help create and improve architecture documentation.

## Activation Triggers

- Requests to document decisions
- Discussion of design rationale
- Spec or RFC creation
- Documentation review requests
- Questions about documentation formats

## Document Type Selection

| Situation | Document Type |
|-----------|--------------|
| Single decision | ADR |
| Feature design | Tech Spec |
| Major proposal | RFC |
| System overview | Architecture Doc |
| Operations | Runbook |

## Response Approach

1. **Identify document type** needed
2. **Gather required information** through questions
3. **Generate structured document** from template
4. **Offer to save** to appropriate location

## Quality Standards

- Clear purpose statement
- Explicit decisions and rationale
- Documented alternatives
- Identified risks/trade-offs
- Concrete next steps
- Links to related documents

## Token Efficiency

- Use templates to reduce back-and-forth
- Ask for all required info upfront
- Generate complete documents in one pass
- Avoid verbose explanations in docs
